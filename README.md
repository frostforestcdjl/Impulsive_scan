# Impulsive_scan

## 1. How to Run the Script?
* Open a terminal, navigate to the impulsive_scan folder, and run: python main.py

## 2. What Happens After Running the Script?
*	A folder named catalog will be created, containing catalogs of individual P-wave detections.
*	A folder named img will be generated. Inside it, station-specific subfolders will be created, each containing waveform images with detected P-wave candidates.

## 3. Can I Specify a Station and Time Span for Detecting Potential P-Wave Events?
* Yes, you can modify the scan parameters in config.py.

## 4. Can I Specify the Time Window for Waveform Images?
* Yes, you can adjust the waveform parameters in config.py.
